Twitter US Airline Sentiment Analysis Dashboard – Power BI

This project analyzes customer sentiment toward major US airlines using Twitter data. Leveraging **Power BI**, the project performs data cleaning, creates interactive visuals, and builds an intuitive dashboard for **real-time sentiment monitoring**.

Features
Sentiment Analysis: Distribution of positive, negative, and neutral sentiments.  
Airline Performance: Compare sentiment trends across major US airlines.  
Time Trends: Analyze sentiment evolution over time.  
Live Tweet Review: Explore actual tweet content for context.  
Interactive Visuals: KPI cards, slicers, bar charts, pie charts, line charts, and a tweet table.  
Dynamic Filters: Focus on specific airlines or sentiment types.

Insights
Identify customer satisfaction levels.  
Detect major complaint patterns.  
Support airline decision-making with actionable insights.

Dataset
The dataset used is publicly available and contains tweets related to major US airlines. You can download it from [Kaggle: Twitter US Airline Sentiment](https://www.kaggle.com/datasets/crowdflower/twitter-airline-sentiment) (replace this link if you used a different source).

Tools & Technologies
Power BI – Data visualization and dashboard creation  
CSV dataset – Source of tweets  

How to Use
1. Download the dataset from the link above.  
2. Open Power BI Desktop.  
3. Import the dataset into Power BI.  
4. Explore the interactive dashboard using filters and visuals.
